# @hetri-courses/trill-responses-api-proxy

<p align="center"><code>npm i -g @hetri-courses/trill-responses-api-proxy</code> to install <code>codex-responses-api-proxy</code></p>

This package distributes the prebuilt [Trill Responses API proxy binary](https://github.com/hetri-courses/trill/tree/main/trill-rs/responses-api-proxy) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/codex-responses-api-proxy.js --help
```

Refer to [`trill-rs/responses-api-proxy/README.md`](https://github.com/hetri-courses/trill/blob/main/trill-rs/responses-api-proxy/README.md) for detailed documentation.
